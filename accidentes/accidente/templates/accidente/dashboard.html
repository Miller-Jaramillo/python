Chart.js
Es una libreria de JavaScript que proporciona un conjunto de tipos de graficos, complementos y varias opciones de personalizacion.
Grafico de linea
Un gráfico de líneas es una forma de trazar puntos de datos en una línea.
Grafico de barras
Un gráfico de barras proporciona una forma de mostrar valores de datos representados como barras verticales.
Grafico de anillos
Los gráficos de anillos son excelentes para mostrar las proporciones relacionales entre los datos.
Grafico circular
Los gráficos circulares son excelentes para mostrar las proporciones relacionales entre los datos.
